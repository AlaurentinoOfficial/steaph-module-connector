# Steaph Module Connector

## Requirements

* Docker (Optinal)
* Mongo v2.6.10 in :27017
* Node.js v9

## Target environemnt

* Start up Steaph Thing Connector
```
$ docker-compose up --build
```

## Development environment

* Start up MongoDB Server
```
$ mongod
```

* Start up Babel listener
```
$ npm install
$ npm run babelw
```

* Start up Steaph API
```
$ nodemon
```
